**Finalité** : choisir et défendre le statut juridique optimal pour Brasse-Bouillon par phase (an 1 → an 3+), avec matrice multi-critères visuelle, scoring pondéré, et triggers de bascule documentés. Slide-ready pour S13 et Q&A "Pourquoi micro et pas SAS dès le départ ?".

**Périmètre** : 5 régimes analysés — Micro-entreprise, EURL, SASU, Portage salarial, Association loi 1901.

## 2. Présentation des 5 régimes

### 2.1 — Micro-entreprise (auto-entrepreneur)

| Paramètre | Valeur 2026 |
|---|---|
| Capital minimum | 0 € |
| Plafond CA (services) | **77 700 € HT/an** |
| Charges sociales | **~22 % sur CA encaissé** |
| Cotisation minimale | 0 € (si CA = 0, pas de charges) |
| Franchise TVA | Oui jusqu'à 36 800 € (seuil bas 2026) |
| Comptabilité | Registre recettes simple (tableur suffit) |
| Coût annuel hors charges | **< 200 €** (CFE uniquement, exonérée an 1) |
| Délai création | **24 h** (formulaire INPI en ligne) |
| Associés possibles | **Non** (personne physique seule) |
| Responsabilité | Limitée au patrimoine professionnel (protection ORIAS) |
| Protection sociale | TNS — retraite de base + complémentaire réduite, **pas d'assurance chômage** |
| ACRE compatible | **Oui** — exonération partielle 12 mois (50 % charges an 1) |

### 2.2 — EURL (Entreprise Unipersonnelle à Responsabilité Limitée)

| Paramètre | Valeur 2026 |
|---|---|
| Capital minimum | 1 € symbolique |
| Plafond CA | **Illimité** |
| Charges sociales gérant TNS | **~25–45 % de la rémunération** |
| Cotisations minimales | ~**1 100 €/an** (même si rémunération = 0) |
| IS | 15 % jusqu'à 42 500 € de bénéfice (taux PME) |
| Comptabilité | Bilan annuel obligatoire — **comptable requis** |
| Coût annuel (compta + charges fixes) | **~2 500–4 000 €** |
| Délai création | **2–4 semaines** (immatriculation INPI + publicité légale) |
| Associés possibles | Non directement (mais transformable en SARL) |
| Responsabilité | **Limitée à l'apport** — patrimoine personnel protégé |
| Protection sociale | TNS — identique micro mais assiette réelle, cotisations minimales |
| ACRE compatible | **Oui** |

### 2.3 — SASU (Société par Actions Simplifiée Unipersonnelle)

| Paramètre | Valeur 2026 |
|---|---|
| Capital minimum | 1 € symbolique |
| Plafond CA | **Illimité** |
| Charges sociales président (assimilé salarié) | **~65–75 % de la rémunération brute** (patronal + salarial) |
| Cotisations minimales | **0 €** (si pas de rémunération — uniquement dividendes) |
| IS | 15 % jusqu'à 42 500 € puis **25 %** |
| Dividendes | 30 % flat tax (PFU — Prélèvement Forfaitaire Unique) |
| Comptabilité | Bilan annuel + dépôt greffe — **comptable requis** |
| Coût annuel (compta + charges fixes) | **~3 000–5 000 €** |
| Délai création | **2–5 semaines** |
| Associés possibles | **Oui** — SASU → SAS en 1 acte notarié simple |
| Responsabilité | **Limitée à l'apport** |
| Protection sociale | **Assimilé salarié** — retraite complémentaire, maladie, **chômage partiel possible** |
| ACRE compatible | **Oui** |

### 2.4 — Portage salarial

| Paramètre | Valeur 2026 |
|---|---|
| Capital minimum | 0 € |
| Plafond CA | Aucun (mais modèle prestation) |
| Charges totales | **~45–50 % du CA** (frais de portage 5–12 % + charges salariales) |
| Cotisations minimales | 0 € (paye uniquement si missions) |
| Comptabilité | **Nulle** — gérée par la société de portage |
| Coût annuel fixe | **0 €** si pas de mission |
| Délai | **Immédiat** — signature contrat |
| Associés possibles | **Non** — salariat |
| Responsabilité | Limitée (salarié protégé) |
| Protection sociale | **Complète** — identique salarié (assurance chômage, maladie, retraite) |
| Pertinence BB | Outil B2B services IA (web studio) — **écarté depuis D3** |

### 2.5 — Association loi 1901

| Paramètre | Valeur 2026 |
|---|---|
| Capital minimum | 0 € |
| Plafond CA | Aucun (mais activité lucrative = risque requalification) |
| Charges sociales | ~45 % sur salariés (si employeur) |
| IS | **Exonéré** si gestion désintéressée et non lucrative |
| Comptabilité | Simple si < 153 000 € de dons/an |
| Coût annuel fixe | ~200 € (déclaration Journal Officiel) |
| Délai création | **48 h** (déclaration en préfecture) |
| Associés possibles | **Oui** — membres, conseil d'administration |
| Responsabilité | Limitée pour les membres (sauf faute de gestion) |
| Protection sociale | Dépend du rôle (bénévole = aucune, salarié = complète) |
| Pertinence BB | Modèle open-source communautaire — **incompatible avec SaaS monétisé** |

## 6. Justification du choix Micro an 1 — défense jury

### 6.1 — Ce que la micro-entreprise nous donne qu'aucune autre ne donne

```
AVANTAGE DÉCISIF MICRO vs CONCURRENTS
──────────────────────────────────────
                    Micro   EURL   SASU
Coût si CA = 0 €     0 €   1100 €  0 €*
Coût si CA = 5K€    1100€  1100 €  1800 €**
Coût si CA = 14K€   3080€  5200 €  7600 €**
Délai création        24h   3 sem   4 sem
Comptabilité requis   Non    Oui     Oui

* SASU : cotisations minimales = 0 si pas de rémunération
  mais IS + frais comptable = ~2500-3000€ fixes
** SASU inclut frais compta 3000€/an
```

**Conclusion** : en phase d'amorçage à CA incertain, chaque euro de charge fixe est un risque mortel pour le projet. La micro-entreprise est la **seule structure à coût strictement proportionnel au CA**.

### 6.2 — Ce que la micro-entreprise ne nous donne pas (assumé)

| Limitation | Impact BB an 1 | Mitigation |
|---|---|---|
| Pas d'associés | Pas de co-fondateurs formels | Acceptable — équipe reste étudiante en an 1 |
| Pas de levée de fonds | Pas d'investisseur external | Assumé — auto-financement documenté (LTD + freemium + LHBS) |
| Pas de protection chômage | Risque fondateur | Atténué : statut étudiant + CPAM encore actif |
| Pas de déduction frais réels | Charges sur CA brut | Optimisable ACRE an 1 (50 % exonération) |
| Image moins "pro" | Perception jury | Contrebalancé par : CI pro + 647 tests + INPI + pitch |

## 7. EURL écarté — pourquoi pas le "compromis"

L'EURL est souvent présentée comme le compromis entre micro et SASU. Elle est en réalité **le pire des deux mondes** pour BB :

```
MICRO                          EURL                          SASU
  │                              │                             │
  ├── Coût création : 0 €        ├── Coût création : ~500 €   ├── Coût création : ~500 €
  ├── Charges : 22 % du CA       ├── Charges : 25-45 %        ├── Charges : 65-75 %
  ├── Compta : aucune            ├── Compta : obligatoire      ├── Compta : obligatoire
  ├── Cotis. min. : 0 €          ├── Cotis. min. : 1100 €/an  ├── Cotis. min. : 0 €*
  ├── Investisseurs : non        ├── Investisseurs : difficile ├── Investisseurs : oui
  └── Protection sociale : faible└── Protection sociale : TNS └── Protection sociale : ✅
```

*Si président sans rémunération.

**EURL coûte plus cher que micro (compta + cotis. min.) sans apporter la protection sociale complète de la SASU ni l'ouverture actionnariat. Elle est optimale uniquement pour un freelance solo avec CA > 50K€ qui ne veut pas d'associés et préfère être TNS.**

## 8. Impact de la suppression du studio web (D3) sur le choix juridique

**Avant D3** : le studio web justifiait la micro-entreprise sur deux jambes :
1. Activité SaaS BB (services informatiques)
2. Prestations web pour TPE food/boissons (consultancy)

La double jambe augmentait la vitesse d'atteinte du plafond micro (77 700 €) → bascule SASU accélérée.

**Après D3** : une seule jambe — SaaS BB uniquement.

**Impact** : le plafond de 77 700 € est atteint plus tard selon le scénario médian (an 2-3 vs an 1-2). La micro-entreprise reste suffisante plus longtemps. **Le choix micro an 1 est encore plus justifié post-D3.**

## 9. Narratif pitch — 20 secondes (Q&A "Pourquoi pas SAS dès le départ ?")

> *"Parce que la micro-entreprise est la seule structure à coût strictement proportionnel au chiffre d'affaires. À CA zéro, charges zéro. À 14 000 euros de revenus an 1, 3 000 euros de charges — contre 7 000 à 9 000 pour une SASU avec comptable. Et on a documenté trois triggers précis de bascule obligatoire vers SASU : CA supérieur à quarante mille euros, premier recrutement, ou levée de fonds. Ce n'est pas une décision par défaut, c'est de l'optimisation d'amorçage."*
