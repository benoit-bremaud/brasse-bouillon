# UI Component Assets – Brasse-Bouillon

This folder contains all graphical assets, design tokens, and code snippets related to core UI components of the Brasse-Bouillon project.

### Structure:

- **buttons/** → Icons, CSS variables, or design files for button components.
- **inputs/** → Assets for input fields (borders, icons, states).
- **alerts/** → Icons or design tokens for alerts & notifications.
- **tooltips/** → Tooltip design files and tokens.

These resources are linked to the "UI Component Styles" section of the `design-charter.md`.
