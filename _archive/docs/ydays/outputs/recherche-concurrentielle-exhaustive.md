## 2. Événements clés 2024-2025 (nouveautés vs competitive-deep-dive)

### 2.1 — Joliebulle FERMÉ (⚡ événement critique)

**Statut** : site joliebulle.org inaccessible depuis début 2025. Maintenance terminée.

**Impact pour Brasse-Bouillon** :
- Joliebulle était le **seul logiciel desktop FR open-source** avec interface en français complète
- Sa fermeture crée un **vide de l'outil gratuit FR** — base d'utilisateurs désormais sans solution
- Timing parfait : BB arrive au moment où l'écosystème FR perd un acteur

**Chiffre à retenir** : Joliebulle comptait une communauté active sur BrassageAmateur.com (~forum threads encore visibles). Utilisateurs orphelins → cible d'acquisition directe pour BB.

**Narratif pitch** : *"En France, Joliebulle, le dernier logiciel gratuit en français, a fermé début 2025. Little Bock existe mais web uniquement, sans scanner. Aucun outil mobile-native en français n'existe aujourd'hui — c'est le vide que Brasse-Bouillon comble."*

### 2.2 — Fermentis App v2 (concurrent surprise)

**Statut** : Fermentis (filiale d'Lesaffre, leader mondial levures brassicoles) a publié la v2 de son app en 2024.

**Périmètre de l'app** : uniquement les levures — calculateur de réhydratation, profil de fermentation, conseils de température. **Pas un gestionnaire de recettes, pas un suivi de brassin.**

**Positionnement** : outil B2B2C — Fermentis le distribue pour vendre ses levures SafAle/SafLager, pas pour monétiser l'app. Aucune fonctionnalité recette, aucun scan.

**Conclusion** : **concurrent indirect uniquement** sur l'étape levure. Brasseur qui utilise Fermentis App a toujours besoin d'un outil de recettes comme BB. Potentielle **complémentarité** (pas adversité).

### 2.3 — BeerSmith 4 (sortie mars 2026)

**Déjà documenté dans competitive-deep-dive §A.2.** Refonte SQL + cloud. Mobile toujours mal noté (3.0/5 iOS). Aucune amélioration UI significative annoncée. Migration vers cloud partielle. **Pas de menace directe sur le segment débutant FR**.

### 2.4 — Brewfather AI Assistant (Premium Plus)

**Fonctionnalité lancée 2025** : assistant IA intégré au tier Premium Plus (~39,99 $/an vs 29,99 $ standard). Suggère des houblons de substitution, ajuste les profils d'eau, répond aux questions techniques en chat.

**Positionnement** : conforte Brewfather dans le segment **expert international**. Creuse l'écart avec les débutants (complexité perçue accrue). **Opportunité indirecte pour BB** qui reste simple par défaut.
