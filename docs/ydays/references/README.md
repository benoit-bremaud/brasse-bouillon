## Règles d'utilisation

- **Ne pas modifier** les documents déposés — ils font foi.
- Si une transcription est utile (email texte → markdown), la nommer
  `<date>_<sujet>.md` et citer l'email source en en-tête.
- Les PDF / images peuvent être commités tels quels (le repo n'a pas
  LFS ; attention à la taille > 1 Mo, prévenir le user avant).
- Les **informations sensibles** (emails personnels du coach,
  numéros de téléphone privés) peuvent être rédigées — utiliser
  `[rédigé pour git]` à la place.
