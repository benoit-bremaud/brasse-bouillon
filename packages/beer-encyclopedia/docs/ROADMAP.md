# Brasse-Bouillon Beer Label AI Roadmap

## Phase 1 — PoC
- Build data collection and annotation workflow
- Train YOLOv8n for label detection
- Add EasyOCR and minimal JSON extraction output

## Phase 2 — MVP
- Add `/scan` endpoint (image upload -> extraction -> recommendation)
- Deliver explainable top-3 recipe ranking
- Add basic logs, low-confidence handling, and smoke tests

## Phase 3 — V1
- Improve dataset quality and model performance
- Capture user feedback to improve recommendation ranking
- Improve latency and observability
