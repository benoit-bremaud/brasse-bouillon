# Cline Rules — Brasse-Bouillon Website

This file is the **primary instruction source** for Cline in this repository.

## Roadmap synchronization policy

1. The website roadmap must be updated **primarily through automated ingest** from `repository_dispatch` events (`roadmap_user_facing_update`).
2. The canonical machine-readable source is `docs/roadmap-feed.json`.
3. `docs/ROADMAP.md` Done table is generated/updated by `scripts/roadmap_sync.py` between markers:
   - `<!-- ROADMAP_DONE_TABLE_START -->`
   - `<!-- ROADMAP_DONE_TABLE_END -->`
4. Manual edits inside those markers are exceptional and must be justified in the PR description.

## Documentation policy

- Keep GitHub-facing documentation in English.
- Keep `README.md`, `CONTRIBUTING.md`, `docs/ROADMAP.md`, and `docs/GOVERNANCE.md` consistent with the actual repository setup.

## Safety rules

- Do not remove roadmap markers from `docs/ROADMAP.md`.
- Do not introduce external Python dependencies for ingestion scripts.
- Keep workflows on least-privilege permissions.
