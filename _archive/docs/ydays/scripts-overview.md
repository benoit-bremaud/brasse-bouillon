# Scripts

Cette section regroupe les scripts du pitch dans l'ordre de passage.

## Lecture recommandée

1. [Bloc 1 — cadrage](/outputs/pitch-script-bloc1-cadrage)
2. [Bloc 2 — avant brassage](/outputs/pitch-script-bloc2-avant-brassage)
3. [Bloc 3 — démo live](/outputs/pitch-script-bloc3-demo-live)
4. [Bloc 4 — après brassage](/outputs/pitch-script-bloc4-apres-brassage)
5. [Bloc 5 — business model + perspectives](/outputs/pitch-script-bloc5-bm-perspectives)
6. [Bloc 6 — conclusion](/outputs/pitch-script-bloc6-conclusion)
7. [Transitions](/outputs/pitch-transitions)

## À regarder en priorité

- le **chrono réel** de chaque bloc ;
- les transitions qui peuvent sonner trop écrites ;
- la cohérence entre les scripts et les slides ;
- les formulations qui paraissent trop agressives ou trop fragiles.

::: warning
Le script le plus sensible à verrouiller reste le
[bloc 3 — démo live](/outputs/pitch-script-bloc3-demo-live), parce qu'il
dépend à la fois du seed, du matériel et du rythme oral.
:::
